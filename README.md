# Course Catalog Test Task

##  Description
A responsive course catalog with search, category filtering and load more functionality.

##  Tech Stack
- HTML5
- SCSS (BEM methodology)
- Vanilla JavaScript (ES Modules)
- Vite (or your bundler)

## Features
- Live search by course title
- Filtering by categories
- Load more pagination
- Responsive layout (desktop → mobile 320px)
- Pixel-perfect layout based on Figma
